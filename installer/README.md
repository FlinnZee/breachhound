# BreachHound installer

Produces `dist\breachhound-setup-<version>.exe` — a Windows installer that
installs the desktop app with Start Menu and optional desktop shortcuts and a
matching uninstaller. It installs per-user by default (no admin needed to
install); the app self-elevates when a scan needs Administrator.

## Prerequisites

- Go 1.26+
- A CGO C compiler (mingw-w64 `gcc`) — Fyne needs it
- `rsrc`: `go install github.com/akavel/rsrc@latest`
- Inno Setup 6 (`ISCC.exe`): `winget install JRSoftware.InnoSetup`

## Build

From the repo root:

```powershell
pwsh scripts\build-installer.ps1
```

or manually:

```powershell
go run ./tools/mkicon
rsrc -ico cmd/breachhound-gui/assets/icon.ico -o cmd/breachhound-gui/rsrc_windows.syso
go build -ldflags "-H windowsgui -s -w" -o dist/breachhound-gui.exe ./cmd/breachhound-gui
ISCC installer\breachhound.iss
```

The app icon is generated by `tools/mkicon` (PNG + multi-size ICO);
`breachhound.iss` defines the installer itself.
